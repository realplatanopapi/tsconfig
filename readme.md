# @realplatanopapi/tsconfig

Resuable tsconfig for TypeScript projects.

## Install

```bash
yarn add @realplatanopapi/tsconfig -D

# or

npm install @realplatanopapi/tsconfig -D
```

## Usage

```javascript
// tsconfig.json
{
  "extends": "@realplatanopapi/tsconfig",
  "compilerOptions": {
    "jsx": "preserve"
  },
  "include": [
    "src"
  ]
}
```
