# @restlessbit/tsconfig

Resuable tsconfig for TypeScript projects.

## Install

```bash
yarn add @restlessbit/tsconfig -D

# or

npm install @restlessbit/tsconfig -D
```

## Usage

```javascript
// tsconfig.json
{
  "extends": "@restlessbit/tsconfig",
  "compilerOptions": {
    "jsx": "preserve"
  },
  "include": [
    "src"
  ]
}
```
